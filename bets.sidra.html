Descrição dos parâmetros

  • x: Número da série

  • from: String ou vetor de caracter specificando a data de início da série (padrão é 1980)

  • to: String ou vetor de caracter specificando a data de fim da série (padrão é ano atual)

  • territory: Especifica o nível territorial da pesquisa

  • variable: Um valor com o código da variável desejada (padrão é selecionar todas as existentes)

  • cl: Vetor contendo as classificações desejadas

  • sections: Vetor ou lista de vetores (se houver mais de duas classificações) contendo as tabelas desejadas

Pesquisa Industrial Mensal - Produção Física

sidra=ecoseries::series_sidra(x = c(3653), from = c("200201"), 
                   to = c("201512"), territory = "brazil", 
                   variable = 3135, sections = c(129316,129330),
