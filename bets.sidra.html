Descrio dos parmetros

  • x: Nmero da srie

  • from: String ou vetor de caracter specificando a data de incio da srie (padro 1980)

  • to: String ou vetor de caracter specificando a data de fim da srie (padro ano atual)

  • territory: Especifica o nvel territorial da pesquisa

  • variable: Um valor com o cdigo da varivel desejada (padro selecionar todas as existentes)

  • cl: Vetor contendo as classificaes desejadas

  • sections: Vetor ou lista de vetores (se houver mais de duas classificaes) contendo as tabelas desejadas

Pesquisa Industrial Mensal - Produo Fsica

sidra=ecoseries::series_sidra(x = c(3653), from = c("200201"), 
                   to = c("201512"), territory = "brazil", 
                   variable = 3135, sections = c(129316,129330),
                   cl = 544)
